# 考研数学历年真题题库系统使用说明

## 🎯 系统概述

本系统是一个考研数学历年真题题库系统，类似zhentiqiang.com的方式呈现，支持数学一、数学二、数学三的历年真题浏览和刷题功能。系统包含完整的真题题库、答案解析、标记功能和数据统计等功能，为考研数学复习提供全面的支持。

## ✅ 功能特点

### 完整题库
- **数学一**: Group 8_1 (17套试卷，386题)
- **数学二**: Group 9_2 (17套试卷，386题)  
- **数学三**: Group 10_3 (17套试卷，386题)
- **年份范围**: 2009-2025年

### 核心功能
1. **年份时间轴**：按2025到2009的顺序组织试卷，横向排列方便浏览
2. **数学类型切换**：支持数学一、数学二、数学三的切换
3. **交互式刷题**：支持按题目顺序刷题，记录答案和标记题目
4. **题目图片显示**：年份框中纵向显示题目图片，确保无缝填充
5. **考点目录**：提供完整的考点目录结构，方便按考点查找题目
6. **夜间模式**：支持夜间模式，保护眼睛
7. **数据持久化**：使用localStorage保存答题状态和标记记录
8. **题号导航**：1-23题快速跳转
9. **键盘操作**：左右键快速切换题目
10. **答案查看**：支持查看对应题目的答案
11. **解析查看**：支持查看对应题目的解析，解析图片直接显示在题目下方
12. **答案图片分类**：按数学类型和年份分类整理答案图片
13. **标记功能**：支持标记题目、查看标记、导出标记和清空标记
14. **标记数据管理**：按年份和数学类型分组管理标记数据
15. **导出标记**：支持导出标记数据为文本文件
16. **解析图片修复**：修复了部分年份无法显示解析图片的问题，所有年份的解析图片都能正常显示

## 📖 使用教程

### 1. 浏览真题

- **数学类型选择**：在顶部导航栏的下拉菜单中选择数学一、数学二或数学三
- **年份选择**：点击时间轴上的年份框，查看对应年份的真题
- **图片查看**：年份框中会显示纵向排列的题目图片，点击年份框进入刷题模式

### 2. 刷题模式

- **开始刷题**：在主内容区点击"开始刷题"按钮
- **题目导航**：顶部的数字按钮可以快速跳转到对应题目
- **答案输入**：在答案输入框中输入你的答案
- **标记题目**：点击"标记题目"按钮标记当前题目
- **查看答案**：点击"查看答案"按钮查看对应题目的答案
- **查看解析**：点击"查看解析"按钮查看对应题目的解析，解析图片会直接显示在题目下方
- **上下题切换**：使用"上一题"和"下一题"按钮切换题目
- **提交答案**：完成所有题目后，点击"提交答案"按钮查看结果
- **键盘操作**：使用左右键快速切换题目

### 3. 夜间模式

- **切换模式**：点击顶部导航栏的"夜间模式"按钮
- **模式记忆**：系统会记住你的模式偏好，下次打开时自动应用

### 4. 标记功能

- **标记题目**：在刷题模式中点击"标记题目"按钮，已标记的题目会显示为橙色
- **显示标记**：点击顶部导航栏的"显示标记"按钮，查看当前年份和类型的所有标记题目
- **导出标记**：点击顶部导航栏的"导出标记"按钮，导出所有标记数据为文本文件
- **清空标记**：点击顶部导航栏的"清空标记"按钮，清空当前年份和类型的所有标记
- **标记数据管理**：标记数据按年份和数学类型分组存储，支持跨年份、跨数学类型的标记管理

### 5. 答案下载与分类

- **下载答案**：运行 `快速下载答案.py` 脚本，按回车即可开始下载答案解析图片
- **默认配置**：默认下载 640-1700 的所有答案，保存到 `answer` 文件夹
- **分类整理**：运行 `tools/tool of sort/sort_images.py` 脚本，按数学类型和年份分类整理答案图片
- **查看解析**：在刷题模式中点击"查看解析"按钮，查看对应题目的解析图片

## 💾 数据存储

### 本地存储
所有答题记录和标记自动保存到浏览器 localStorage：

- **答案记录**: `answers_visual`
- **标记题目**: `markedQuestions_visual` (按年份和数学类型分组的对象结构)
- **夜间模式**: `nightMode`

### 标记数据结构

标记数据采用按年份和数学类型分组的对象结构：

```javascript
{
    "数学一_2025": [1, 3, 5, 7],
    "数学二_2024": [2, 4, 6],
    "数学三_2023": [1, 2, 3]
}
```

这种结构支持不同年份和数学类型的独立标记管理，便于用户按年份和类型查看和管理标记题目。

### 清除数据
```javascript
// 在浏览器控制台执行
localStorage.clear()
```

## 常见问题

### Q1: 图片无法显示？
**A**: 检查以下几点：
1. 确认 `题目图片` 文件夹与 `真题墙(双击打开).html` 在同一目录
2. 确认文件夹结构正确：`题目图片/group_X_Y/paper_Z/ZZ.png`
3. 确认图片文件名格式：`01.png`, `02.png` (两位数字)

### Q2: 如何查看所有已答题目？
**A**: 打开浏览器控制台（F12），输入：
```javascript
console.log(JSON.parse(localStorage.getItem('answers_visual')))
```

### Q3: 如何导出答题记录？
**A**: 在控制台执行：
```javascript
const data = {
    answers: JSON.parse(localStorage.getItem('answers_visual')),
    marked: JSON.parse(localStorage.getItem('markedQuestions_visual'))
};
console.log(JSON.stringify(data, null, 2));
```

### Q4: 支持哪些浏览器？
**A**: 支持所有现代浏览器：
- Chrome/Edge (推荐)
- Firefox
- Safari
- Opera

### Q5: 可以离线使用吗？
**A**: 可以！只要题目图片已下载，无需网络即可使用。

### Q6: 解析图片无法显示？
**A**: 系统已修复了解析图片显示问题，所有年份的解析图片都能正常显示。如果仍然无法显示，请检查：
1. 确认 `sorted_images` 文件夹存在且结构正确
2. 确认已运行 `sort_images.py` 脚本对答案图片进行了分类整理
3. 确认浏览器控制台是否有错误信息
